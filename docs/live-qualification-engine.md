# Live qualification engine

During active group matches, CupMarket now separates three views:

1. The original pre-match model forecast.
2. A provisional table that treats current live scores as final for display only.
3. A live projection that fixes the current score and simulates only the minutes still to play.

The engine also evaluates both simultaneous group matches, the best-third-place line, helpful results elsewhere in the group, and the effect of the next possible goal. Official country prices remain unchanged until the normal post-match automation completes.
