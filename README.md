# CupMarket 2026

A Streamlit dashboard for World Cup match tracking, tournament probabilities, and virtual country prices.

## Deploy

1. Upload this folder to a GitHub repository.
2. Open Streamlit Community Cloud.
3. Create an app from the repository.
4. Use `app.py` as the main file.
5. Add this secret in the app settings:

```toml
FOOTBALL_DATA_TOKEN = "YOUR_NEW_TOKEN"
```

Do not commit the token to GitHub.

## Data freshness

- Match scores can refresh every 60 seconds when the token is configured.
- Model probabilities and country prices are generated by the Colab pipeline.
- Run Phases 4 and 5 again, then rebuild this package to publish newer analytics.
